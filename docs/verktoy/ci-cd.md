# 🚀 CI/CD

Teamets oppsett for kontinuerlig integrasjon og leveranse med GitHub Actions.

## Oversikt

Team eSyfo bruker GitHub Actions for bygg, test og deploy til NAIS-plattformen.

## Innhold (planlagt)

- Oppbygging og gjenbruk av GitHub Actions-arbeidsflyter
- Utrulling til dev- og produksjonsmiljø
- Utrulling fra feature-brancher
- Hemmeligheter og miljøvariabler
- Overvåkning av kjøringer
