# 🚀 CI/CD

Teamets oppsett for kontinuerlig integrasjon og leveranse med GitHub Actions.

## Oversikt

Team eSyfo bruker GitHub Actions for bygg, test og deploy til NAIS-plattformen.

## Gjenbrukbare workflows

Teamets felles GitHub Actions-workflows ligger i et eget repo:

**Repo:** [navikt/teamesyfo-github-actions-workflows](https://github.com/navikt/teamesyfo-github-actions-workflows)

Her finner du gjenbrukbare workflows og composite actions som applikasjonene i teamet refererer til. Endringer her påvirker CI/CD-pipelines på tvers av alle repoene våre.

## Innhold (planlagt)

- Utrulling til dev- og produksjonsmiljø
- Utrulling fra feature-brancher
- Hemmeligheter og miljøvariabler
- Overvåkning av kjøringer
