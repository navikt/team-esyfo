# 📋 Kartleggingsspørsmål

**Fase:** Tidlig 🟢

## Formål

Kartleggingsspørsmålene samler inn informasjon fra den sykmeldte tidlig i forløpet. Svarene gir Nav og arbeidsgiver bedre grunnlag for å vurdere hva slags oppfølging som trengs.

## Dette området handler om

- Stille relevante spørsmål til den sykmeldte i tidlig fase
- Samle inn svar som grunnlag for videre oppfølging
- Gi Nav-veiledere og arbeidsgiver innsikt i den sykmeldtes situasjon

## Viktige begreper

| Begrep | Forklaring |
|--------|-----------|
| Kartlegging | Strukturert innhenting av informasjon fra den sykmeldte |
| Tidlig intervensjon | Tiltak som settes inn tidlig for å forebygge langvarig fravær |

## Brukergrupper

- Den sykmeldte
- Nav-veileder
