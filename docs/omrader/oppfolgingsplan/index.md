# 📝 Oppfølgingsplan

**Fase:** Midtveis 🟡

## Formål

Oppfølgingsplanen dokumenterer tiltak og mål for tilbakeføring til arbeid. Planen lages i samarbeid mellom arbeidsgiver og arbeidstaker, og kan deles med Nav og lege.

## Dette området handler om

- La arbeidsgiver og arbeidstaker opprette og redigere en oppfølgingsplan
- Dokumentere arbeidsoppgaver, tilrettelegging og mål
- Dele planen med Nav-veileder og fastlege ved behov

## Viktige begreper

| Begrep | Forklaring |
|--------|-----------|
| Oppfølgingsplan | Skriftlig plan med tiltak, mål og tidsfrister for tilbakeføring |
| Tilrettelegging | Tilpasninger av arbeidssituasjonen for den sykmeldte |
| Deling med Nav | Mulighet for å sende planen til Nav-veileder |

## Brukergrupper

- Arbeidsgiver / nærmeste leder
- Den sykmeldte
- Nav-veileder (mottaker)
- Fastlege (mottaker)
