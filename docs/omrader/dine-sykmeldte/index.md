# 👥 Dine sykmeldte

**Fase:** Gjennomgående 🔵

## Formål

«Dine sykmeldte» gir arbeidsgivere en samlet oversikt over sine sykmeldte ansatte. Appen brukes gjennom hele sykefraværsforløpet og er arbeidsgivers hovedinngang til oppfølging.

## Dette området handler om

- Vise arbeidsgivere hvem som er sykmeldt og status på oppfølging
- Samle relevant informasjon og handlinger for hver sykmeldt ansatt
- Gi varsler og påminnelser om frister og oppgaver

## Viktige begreper

| Begrep | Forklaring |
|--------|-----------|
| Arbeidsgiver | Person med personalansvar som følger opp den sykmeldte |
| Sykmeldt ansatt | Arbeidstaker som har aktiv sykmelding |
| Oppfølgingsstatus | Oversikt over hvor i forløpet oppfølgingen befinner seg |

## Brukergrupper

- Arbeidsgiver / nærmeste leder
