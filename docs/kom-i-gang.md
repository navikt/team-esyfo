:::details Skrivetips
Denne siden bruker VitePress sin [markdown-syntax](https://vitepress.dev/guide/markdown).

```md
:::tip Tips
Kort og nyttig råd.
:::
```

- `:::tip` — nyttige råd
- `:::info` — bakgrunnsinformasjon
- `:::warning` — forutsetninger og advarsler
- `:::danger` — ting som må unngås
- `:::details` — skjulbart innhold
:::
