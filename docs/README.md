# Team eSyfo Wiki

VitePress-basert wiki for Team eSyfo.

**Publisert:** [navikt.github.io/team-esyfo](https://navikt.github.io/team-esyfo/)

## Mise-oppgaver

All wiki-utvikling kjøres via [mise](https://mise.jdx.dev/)-oppgaver definert i [`mise.toml`](../mise.toml):

| Oppgave | Beskrivelse |
|---|---|
| `mise run wiki:install` | Installer avhengigheter med pnpm |
| `mise run wiki:dev` | Start lokal utviklingsserver |
| `mise run wiki:build` | Bygg wikien for produksjon |
