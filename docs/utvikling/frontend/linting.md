# Linting og formatering Frontend

Vi bruker **Biome** som standard verktøy for linting og automatisk formatering av frontend-kode i team eSYFO. Biome kombinerer linting og formatering med fokus på ytelse og enkel konfigurering.

Biome sjekker koden for logiske feil og best practices, samtidig som den formaterer den automatisk for konsistens. Det kjører enkelt som en del av build-prosessen eller fra kommandolinjen.

[Lefthook](https://biomejs.dev/recipes/git-hooks/#lefthook) er brukt som hook manager og er støttet fra Biome.

## Konfigurasjon

Se eksempel på Biome-konfigurasjon i [narmesteleder-frontend](https://github.com/navikt/narmesteleder-frontend/blob/main/biome.json).
